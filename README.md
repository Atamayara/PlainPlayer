## 使い方
起動して`H`を入力するとヘルプがあります。

## 設定
Macの場合は`/Users/<UserName>/Library/Application Support/PlainPlayer/settings.yml`に設定ファイルがあります。
Windowsの場合は分かりません。

以下が設定ファイルの例です。(v.0.1.0現在以下の項目のみ対応)

```yml
Paths:
  Rekordbox: <rekordbox xmlへのパス>
```
